## Stinger Seller

Educational (?) project.

## How do I develop this?

On MacOS/Linux:

```sh
python -m venv venv/
. venv/bin/activate
pip install -r requirements.txt
```

On Windows:
```sh
python -m venv venv/
./venv/Scripts/activate
pip install -r requirements.txt
```

## How do I run this?

Now, just:
```sh
flask --app flaskr init-db
flask --app app run --debug
```
